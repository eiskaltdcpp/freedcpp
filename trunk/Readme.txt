Introduction
------------
FreeDC++ -DC++ клиент основан на исходном коде LinuxDC++
Изменения смотрите в Changelog.txt

Dependencies:
-------------
scons >= 0.96
pkg-config
g++ >= 3.4
gtk+-2.0 >= 2.10
gthread-2.0 >= 2.4
libglade-2.0 >= 2.4
pthread
zlib
libbz2
libssl

Compiling:
----------
$ cd /path/to/freedcpp
$ scons PREFIX=/path/to/install

Installing:
-----------
# scons install

Running:
--------
$ freedcpp
